# Golang-OpenAPI
Go To Daejeon 발표 주제 및 Golang OpenAPI 구현

Golang gin-swagger를 이용하여 RESTapi를 간편하게 사용하기 위해 
Swagger 구현 레포입니다.

## 실행

### docker 

```shell
docker pull ghcr.io/mireu-lab/golang-openapi:main
```

```shell
docker run -p [export port]:80 ghcr.io/mireu-lab/golang-openapi:main
```

### Shell

```shell
go run run/sample
```
